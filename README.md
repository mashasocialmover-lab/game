# Game Project

Проект игры "Cats vs Dogs: HP Bars Battle"

## Автоматическая загрузка на GitHub

Проект настроен для автоматической загрузки изменений на GitHub.

### Настройка авторизации

**ВАЖНО**: Токен, который вы предоставили, не работает. Нужно:

1. **Вариант 1 (РЕКОМЕНДУЕТСЯ)**: Используйте GitHub Desktop
   - Откройте GitHub Desktop
   - Войдите в свой аккаунт через браузер
   - Все изменения будут автоматически синхронизироваться

2. **Вариант 2**: Создайте новый токен с правильными правами
   - Перейдите на https://github.com/settings/tokens
   - Создайте новый токен (classic)
   - Обязательно отметьте право **repo** (полный доступ к репозиториям)
   - Используйте токен в формате: `x-access-token:ВАШ_ТОКЕН`

### Как это работает

1. **Git Hook (post-commit)**: Автоматически отправляет изменения на GitHub после каждого коммита
2. **Скрипт auto-push.ps1**: Можно запустить вручную для отправки изменений

### Использование

После настройки авторизации, все изменения будут автоматически загружаться на GitHub при коммитах.

Для ручной отправки:
```powershell
.\auto-push.ps1
```
