# Инструкция по настройке авторизации GitHub

## Вариант 1: Авторизация через браузер (РЕКОМЕНДУЕТСЯ)

1. В окне GitHub Desktop выберите "Sign in with your browser"
2. В браузере войдите в свой аккаунт GitHub
3. Подтвердите доступ для GitHub Desktop
4. Готово! Авторизация настроена автоматически

## Вариант 2: Создание Personal Access Token (PAT)

Если нужно использовать токен:

1. Перейдите на https://github.com/settings/tokens
2. Нажмите "Generate new token" → "Generate new token (classic)"
3. Дайте токену имя (например, "Game Project Auto Push")
4. Выберите срок действия (expiration)
5. Отметьте права (scopes):
   - ✅ **repo** (полный доступ к репозиториям)
6. Нажмите "Generate token"
7. **ВАЖНО**: Скопируйте токен сразу! Он показывается только один раз
8. Сохраните токен в безопасном месте

### Использование токена:

После создания токена, его можно использовать:
- В GitHub Desktop: Settings → Accounts → Add Account → Token
- Или в командной строке git (будет настроено автоматически)

## После авторизации

После успешной авторизации все изменения будут автоматически загружаться на GitHub!
